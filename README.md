# BrainWords

A small game inspired by a childhood classic, but this time… with zombies. 

⚠️ This project uses a local API (http://localhost:3333) Make sure to run the API server before starting the game.

### Installation

Install the dependencies:

```bash
npm install
```
Go on the API:

```bash
cd node-hangman-api
```

and install his dependencies :

```bash
npm install
```

### Development Or Play

Start the development server with HMR:

```bash
npm run dev
```

But do the same for the api befor starting, so at first do :

```bash
cd node-hangman-api
```

And : 

```bash
npm run dev
```

Your application will be available at `http://localhost:5173`.
